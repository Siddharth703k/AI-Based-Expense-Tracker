# 💰 AI-Based Expense Tracker

An intelligent expense tracking web application built using **Flask + SQLite + Python**, with built-in **AI-driven insights** to help users manage spending habits, detect anomalies, and optimize budgets.

---

## 🚀 Features

### 📊 Expense Management
- Add, view, and delete expenses
- Categorize expenses (Food, Travel, Shopping, Custom)
- Track spending with date and description

### 🎯 Budget Tracking
- Set a monthly budget
- View remaining balance in real-time
- Visual category-wise spending breakdown

### 🤖 AI-Powered Insights
The app includes intelligent analytics such as:
- 📅 Daily spending summary
- 🔥 Budget burn rate prediction
- 📈 Monthly spending projection
- ⚠️ Anomaly detection (unusual high expenses)
- 📊 Category imbalance detection
- 🧠 Spending pattern analysis (weekday vs weekend)
- 🏆 Weekly improvement tracking
- 💡 Smart financial tips

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS (Custom UI)
- **AI Logic:** Python (Statistics & Rule-Based Intelligence)
